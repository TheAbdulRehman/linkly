# Bookmark Manager (name TBD)

A bookmark management application. The project name is not finalized yet.

## Overview

This is the backend for a bookmark management app — a place to save, organize, and manage your bookmarks. The repo currently holds the backend; the frontend lives separately.

## Tech Stack

### Backend
- **Node.js** with **Express** — REST API server
- **PostgreSQL** — relational database
- **Neon** — serverless Postgres hosting

### Frontend
- **Next.js** — React framework
- **React** — UI library
- **React Query** — server state management & data fetching
- **Tailwind CSS** — styling
- **shadcn/ui** — component library

## Architecture

The codebase follows a **feature-based structure**, where code is organized by feature/domain rather than by technical layer.
